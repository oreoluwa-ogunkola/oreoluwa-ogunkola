Predicting House Prices¶

The housing price prediction project aims to develop a machine learning model that accurately estimates residential property prices based on various relevant features. By leveraging a combination of Python libraries, including pandas, seaborn, matplotlib.pyplot, category_encoders, and scikit-learn, this data-driven project seeks to provide valuable insights into the real estate market and support informed decision-making for buyers, sellers, and investors.

The dataset includes several critical attributes that influence property prices, such as 'area' (the total square footage of the property), 'bedrooms' (the number of bedrooms in the house), 'bathrooms' (the count of bathrooms), 'stories' (the number of floors in the building), and binary categorical variables like 'mainroad,' 'guestroom,' 'basement,' 'hotwaterheating,' 'airconditioning,' 'parking,' 'prefarea,' and 'furnishingstatus.' Each feature will be utilized to understand its impact on the target variable, i.e., the housing price.

The process involves data preprocessing, exploration, and feature engineering to handle missing values, outliers, and categorical variables. One-hot encoding will be used to transform categorical features into numerical representations, making them suitable for training various machine learning models.

Three different regression algorithms - Linear Regression, Ridge Regression - will be employed to build predictive models. By splitting the dataset into training and testing sets using train_test_split from scikit-learn, the models' performance will be evaluated based on the Mean Absolute Error (MAE) metric, which measures the accuracy of predictions.

The primary objective of this project is to develop a robust and accurate housing price prediction model that can provide reliable estimates based on the given features. This model can serve as a valuable tool for real estate professionals, potential buyers, and property investors looking to make well-informed decisions and maximize their returns in the real estate market.

data source: https://www.kaggle.com/datasets/harishkumardatalab/housing-price-prediction

# build a wrangle function to read the data

def wrangle(filepath):
    # Read CSV file
    df = pd.read_csv(filepath)
    
    # Subset data: Remove outliers for area
    low,high =df["area"].quantile([0,0.9])
    mask_area = df["area"].between(low,high)
    df = df[mask_area]
    
    return df

Collinearity, also known as multicollinearity, is a statistical term that refers to a high correlation or linear relationship between two or more predictor variables (also known as independent variables) in a regression model. In other words, it occurs when two or more independent variables are closely related, making it challenging for the regression model to distinguish their individual effects on the dependent variable.

We are going to check for collinearity by calculating the correlation of the feature variables and draw a heatmap for readability
