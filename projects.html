Analysis of Playstore data

The housing price prediction project utilizes Python libraries like pandas, seaborn, and scikit-learn to develop a machine learning model for estimating residential property prices. The dataset comprises features like 'area,' 'bedrooms,' 'bathrooms,' and binary categorical variables like 'mainroad' and 'airconditioning.' Data preprocessing and feature engineering are employed to handle missing values and transform categorical variables using one-hot encoding. Three regression algorithms, including Linear Regression and Ridge Regression, are used to build predictive models. The model's accuracy is evaluated using Mean Absolute Error (MAE), aiming to provide accurate estimates for housing prices and support decision-making in the real estate market.
